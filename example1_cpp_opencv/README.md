# example1_cpp_opencv: ImageProcessor

This is a simple example of a C++ project that uses OpenCV to process images. The project is built using CMake.

## Dependencies 依赖

- OpenCV

    ```bash
    sudo apt-get install libopencv-dev
    ```

## Build and Run 编译和运行

```bash
cmake -B build
make -C build
```

```bash
./build/main
```
